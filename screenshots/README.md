# Screenshots folder
This folder contains screenshots of the Resume Classification System application.

## How to add screenshots:

1. Save your screenshots as PNG files
2. Name them as follows:
   - `1_main_interface.png` - Main app interface
   - `2_file_upload.png` - File upload example
   - `3_classification_results.png` - Classification results
   - `4_sample_resumes.png` - Sample resumes tab
   - `5_ai_classification.png` - AI classification example

3. Place them in this folder and commit to Git
